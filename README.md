# Adventure Park Website
A static multi-page web platform designed for a modern Adventure Park, engineered from the ground up using HTML5 and CSS3. This project demonstrates strong foundational skills in semantic document mapping, stylesheet architecture, and UI layout design.

## 🎨 Core Implementation
* **Thematic UI Design:** Developed a cohesive visual aesthetic featuring high-energy colour palettes and custom typography suited for an adventure tourism platform.
* **Component-Based Styling:** Implemented structured CSS rules to manage navigation menus, hero sections, pricing grids, and informational modules.
* **Semantic Document Architecture:** Built clean HTML5 structures using meaningful tags to ensure readable document outlines and logical content flows.
* **Asset Organization:** Structured project directories efficiently, maintaining separate, clean files for different pages and style assets.

## 🛠️ Tech Stack & Concepts
* **HTML5:** Semantic markup, form structures, and page routing via anchor links.
* **CSS3:** Box model manipulation, structural positioning, text styling, and custom hover states.
